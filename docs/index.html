🎯 Project Overview

This project serves as a template and reference for students learning Object-Oriented Programming (OOP) and Software Quality. It simulates a battleship environment where players must strategically place ships and sink the enemy fleet.

The Rules

The game is played on a grid (typically 10×10). Players take turns calling coordinates to fire upon the enemy grid. The first to sink all enemy ships wins.

🚀 Installation & Setup

Prerequisites

JDK 17+ Git Maven 3.8+
1

Clone the repository

git clone https://github.com/britoeabreu/Battleship2.git
2

Navigate to the project directory

cd Battleship2
3

Compile and run

mvn clean package
java -jar target/BattleshipGamePlayer-2.0.jar
